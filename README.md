# Marauder

Kartbasert PWA for å utforske Harry Potter-universet i Storbritannia.

Live: [elzacka.github.io/marauder-pwa](https://elzacka.github.io/marauder-pwa/)

## Utvikling

```bash
npm install
npm run dev
```

## Datakilder
